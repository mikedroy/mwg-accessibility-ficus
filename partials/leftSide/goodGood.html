Unit 2:
Product Grid

Improvements made

Now that we've fixed our accessibility problems, use your screen reader and keyboard to:
  1. Try to add cheese.
  2. Determine how many items there are.
By using a list for the products, screen readers will typically state the number of items in the list. In order to improve the experience we:
  1. Added alt text to images. Now the images can be read by the screen reader indicating what this image is.
  2. Added an aria-label to the Add buttons. This gives people using screen readers more information about what this button will actually do.
