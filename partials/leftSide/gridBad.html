Unit 2:
Product Grid

Can you find it?

In the pane on the right is an invisible product grid for a fictional store. You can access it with a screen reader like NVDA and your keyboard.

Using those tools, attempt to:
  1. Try to add cheese.
  2. Determine how many items there are.
  3. When you're done attempting to navigate this version of the product grid, click "continue lesson" to move on to phase 2.
