Now You See It

Welcome

This project provides developers and others interested in accessible web development with an informative and hands-on lesson about the challenges that exist for visually impaired users. It also provides useful feedback on inclusive web page design.

Instructions

You will need the NVDA Reader (Windows)

Click here to download NVDA

Installing the NVDA Reader

  1. Agree to the License agreement and choose the option to Install NVDA on this Computer.
  2. Uncheck the option to Use NVDA on the Windows Logon Screen and click Continue.
  3. Click OK after installation to start the installed copy.

Running NVDA

  1. If not already running, locate the NVDA application in the program directory and click to run.
  2. Set desired start options on the first popup and click OK.
  3. Turn on/up the volume and proceed.
